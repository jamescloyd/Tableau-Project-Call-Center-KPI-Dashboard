# Tableau Project: Call Center KPI Dashboard

In this project, I acted as a data analyst in a company where a manager in the call center (help desk) wants a monthly KPI dashboard. He wants to track how his employees are doing as well as some overall information about the calls. As a data analyst, I am to create a dashboard to give him useful information about his data, but he isn't exactly sure what he wants.

In the dataset, there is information about the name of the agent, the date and time of the incoming call, the topic of the call, if the call was answered, if the problem of the customer was solved, the duration of the call, and the satisfaction rating given by the customer, etc.

Out of the dataset, I created a dashboard in which there are charts and graphs that demonstrate the following KPIs (for the latest month in the dataset) such as the call volume of each day during the week, the overall resolution rate, the number of inbound calls, the average duration of the call (in minutes), the average speed of answer by agent, and the resolution rates by agent.

Please find the Tableau dashboard file in this repository or visit [my Tableau Public profile](https://public.tableau.com/views/CallCenterKPIDashboard_17735835617510/Dashboard1?:language=en-GB&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link).
